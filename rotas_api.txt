ROTS API - SmartPlate
===================

AUTENTICAÇÃO (auth)
-------------------

POST /auth/register
Request: 
{
  "name": "string",
  "email": "string", 
  "password": "string"
}

Response (200):
{
  "name": "string",
  "email": "string",
  "token": "string"
}

POST /auth/login
Request:
{
  "email": "string",
  "password": "string"
}

Response (200):
{
  "name": "string",
  "email": "string", 
  "token": "string"
}

REFEIÇÕES DO USUÁRIO (usermeals)
---------------------------------

POST /usermeals/usermeal
Headers: Authorization: Bearer <token>
Request:
{
  "mealName": "string",
  "description": "string",
  "imageBytes": "byte[]"
}

Response (200):
{
  "mealName": "string",
  "description": "string",
  "meal_date": "datetime",
  "meal_time": "timespan",
  "calories": decimal,
  "protein_g": decimal,
  "carbs_g": decimal,
  "fat_g": decimal
}

GET /usermeals/usermeals?date=<datetime>
Headers: Authorization: Bearer <token>
Response (200):
[
  {
    "mealName": "string",
    "description": "string", 
    "meal_date": "datetime",
    "meal_time": "timespan",
    "calories": decimal,
    "protein_g": decimal,
    "carbs_g": decimal,
    "fat_g": decimal
  }
]

ERROS COMUNS
------------

400 Bad Request:
{
  "success": false,
  "message": "string"
}

401 Unauthorized:
Retornado quando o token JWT não é fornecido ou é inválido
